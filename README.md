# AI Trading Project

VS Code Python 확장프로그램과 함께 개발된 주식 분석 프로그램입니다.

## 🚀 기능
- 실시간 주식 가격 조회
- 인기 주식 일괄 조회
- 가격 변동률 계산

## 🛠️ 사용된 VS Code 확장프로그램
- **Python**: Python 언어 지원
- **Pylance**: 인텔리센스 및 타입 검사
- **Python Debugger**: 디버깅 기능
- **Python Environments**: 가상환경 관리

## 📦 설치된 패키지
- yfinance: 주식 데이터 수집
- pandas: 데이터 처리
- numpy: 수치 계산

## 🎯 실행 방법
```bash
python stock_analyzer.py
```

## 🔧 디버그 방법
1. F5 키를 눌러 디버그 모드 실행
2. 브레이크포인트 설정 가능
3. 변수 값 실시간 확인

## 📊 예시 주식 심볼
- AAPL (Apple)
- TSLA (Tesla)
- GOOGL (Google)
- MSFT (Microsoft)
- AMZN (Amazon)
